# 使用说明

## 环境要求

- Linux 系统
- Python 3.10 或更高版本
- bash、grep、awk、sed、df、du、ps、ss 等常见命令

## 安装依赖

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

## 配置 DeepSeek

编辑 `.env`：

```text
DEEPSEEK_API_KEY=你的key
DEEPSEEK_MODEL=deepseek-v4-flash
```

如果没有 API Key，可以使用 `--no-llm`：

```bash
python main.py "检查磁盘空间问题" --no-llm
```

## 常用命令

```bash
python main.py --list-skills
python main.py --skill disk_check
python main.py --skill log_analyze
python main.py "检查磁盘空间问题"
python main.py "分析 SSH 登录失败"
python main.py --report daily
```

## 输出结果

终端会显示诊断报告，同时在 `reports/` 目录保存 Markdown 文件。

## 安全说明

系统不会执行用户或 AI 动态生成的任意 Shell 命令，只会执行项目中预先登记的脚本。
