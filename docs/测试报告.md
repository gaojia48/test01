# 测试报告

## 测试目标

验证 Linux 运维 Agent 的 skill 加载、安全执行、Shell 脚本和报告生成能力。

## 测试命令

```bash
python3 -m unittest discover -s tests
python main.py --list-skills
python main.py --skill disk_check --no-llm
python main.py "检查磁盘空间问题" --no-llm
bash scripts/disk_check.sh
bash scripts/log_analyze.sh
```

## 预期结果

- 所有 skill 能正常加载。
- 执行器只允许运行登记脚本。
- 脚本在普通用户权限下也能输出可解释结果。
- DeepSeek 不可用时，本地关键词匹配仍能选择 skill。
- 报告保存到 `reports/` 目录。

## 异常场景

- API Key 缺失：自动使用本地模式。
- 日志文件不存在：提示可用替代数据源。
- 权限不足：输出权限提示。
- 高风险请求：拒绝执行破坏性操作。
